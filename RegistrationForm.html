Registration Form using
Spring Boot

This is a personal project of mine to develop a basic spring boot web application.
In this project, Spring Data JPA, Spring Web are mainly used.
Web UI templates are from free sources and customized depending on requirements of my project.

Spring data JPA
Takes care of managing database.

It makes creating database entity simple and fast.
Basically, it checks classes which are annotated with @Entity and manage tables in database.

Spring data JPA also makes fetching and managing data (CRUD operation) from the database easier.
No more hard code for queries like JDBC.

Spring web
will take care of web process, similar to JavaEE

It will manage redirecting pages, map functions with request urls.
Basically, REST services and request mapping process.
